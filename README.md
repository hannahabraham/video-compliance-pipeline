# About the Project

This project builds an automated Video Compliance QA Pipeline using LangGraph to check videos against compliance and regulatory rules. It uses Azure Video Indexer to extract transcripts and on-screen text (OCR) from videos, while Azure AI Search and Azure OpenAI Embeddings help retrieve the most relevant compliance policies through a RAG-based system. Azure OpenAI (GPT-4o) analyzes the video content and identifies possible violations in a reliable and structured way. LangSmith is used to trace and optimize the LLM workflow, and Azure Application Insights provides real-time monitoring, logging, and performance tracking. The system converts unstructured video content into clear, actionable JSON compliance reports with complete end-to-end observability.
